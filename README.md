## Term 2 Project - LET LOOSE

![LET LOOSE](https://github.com/EwanCarver/portfolio/blob/main/images/letloosecropped.png?raw=true)<br>
![Gorp](https://github.com/EwanCarver/portfolio/blob/main/images/bigGorp.gif?raw=true)
![Carl](https://github.com/EwanCarver/portfolio/blob/main/images/BigShootySean.gif?raw=true)
![Run](https://github.com/EwanCarver/portfolio/blob/main/images/Runguybig.gif?raw=true)

This game was a group project. I worked with Grace Perry and Gabriel Farley to make this game using processing.

Check out the [GITHUB REPOSITORY](https://eggz-csm.github.io/LET_LOOSE).

I personally made the majority of graphics as well as all of the sound effects and the implementation of them.
I also created a "MusicManage" class that worked to make adaptive music. When in range of an enemy, the music will smoothly transition to a more intense version of the background music. (I also made this music myself)
